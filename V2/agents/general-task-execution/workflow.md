# General Task Execution Agent - Workflow Diagrams

## Основной поток выполнения

```mermaid
flowchart TD
    A[Получение задачи] --> B[Анализ требований]
    B --> C{Достаточно контекста?}
    C -->|Нет| D[Запрос дополнительной информации]
    D --> B
    C -->|Да| E[Планирование выполнения]
    E --> F[Выполнение задачи]
    F --> G{Требуются тесты?}
    G -->|Да| H[Создание/запуск тестов]
    G -->|Нет| I[Валидация результата]
    H --> J{Тесты прошли?}
    J -->|Нет| K[Исправление ошибок]
    K --> H
    J -->|Да| I
    I --> L{Результат корректен?}
    L -->|Нет| M[Корректировка]
    M --> F
    L -->|Да| N[Возврат результата]
```

## Поток создания файла

```mermaid
flowchart TD
    A[Запрос на создание файла] --> B[Определение пути и структуры]
    B --> C[Проверка существования]
    C --> D{Файл существует?}
    D -->|Да| E[Запрос подтверждения перезаписи]
    E --> F{Перезаписать?}
    F -->|Нет| G[Отмена операции]
    F -->|Да| H[Создание файла]
    D -->|Нет| H
    H --> I[Добавление содержимого]
    I --> J{Файл большой?}
    J -->|Да| K[Использование fsWrite + fsAppend]
    J -->|Нет| L[Использование fsWrite]
    K --> M[Валидация синтаксиса]
    L --> M
    M --> N{Ошибки?}
    N -->|Да| O[Исправление]
    O --> M
    N -->|Нет| P[Файл создан]
```

## Сценарий: Создание компонента с тестами

```mermaid
sequenceDiagram
    participant U as User
    participant A as Agent
    participant FS as File System
    participant Test as Test Runner

    U->>A: Создай компонент Button с тестами
    A->>A: Анализ требований
    A->>FS: fsWrite(Button.jsx)
    FS-->>A: Файл создан
    A->>FS: fsWrite(Button.test.jsx)
    FS-->>A: Тесты созданы
    A->>Test: executeBash(npm test)
    Test-->>A: Результаты тестов
    alt Тесты прошли
        A->>U: Компонент создан, тесты проходят
    else Тесты упали
        A->>A: Анализ ошибок
        A->>FS: Исправление кода
        A->>Test: Повторный запуск
        Test-->>A: Результаты
        A->>U: Компонент создан после исправлений
    end
```

## Сценарий: Рефакторинг с переименованием

```mermaid
sequenceDiagram
    participant U as User
    participant A as Agent
    participant Code as Code Tools
    participant Diag as Diagnostics

    U->>A: Отрефактори функцию getUserData
    A->>Code: readCode(файл с функцией)
    Code-->>A: Структура кода
    A->>A: Планирование рефакторинга
    A->>Code: semanticRename(getUserData → fetchUserData)
    Code-->>A: Переименование выполнено
    A->>Code: editCode(улучшение логики)
    Code-->>A: Изменения применены
    A->>Diag: getDiagnostics(измененные файлы)
    Diag-->>A: Нет ошибок
    A->>U: Рефакторинг завершен
```
