это проект Ergodeon - Autonomous code orchestration
Ergodeon - автономный AI-агент на Python для разработки программного обеспечения. Использует LLM через API OpenRouter для понимания инструкций пользователя и взаимодействия с операционной системой.
Ключевая особенность агента - способность планировать, выполнять инструменты (чтение/запись файлов, запуск команд) и поддерживать контекст диалога. При работе над проектом агент автоматически создает документацию (чеклист, walkthrough, план) в папке проекта.

Даю тебе два файла 
Ergodeon_Ergodeon_*_content.md - тут все файлы проекта запакованы в один текстовый файл 
Ergodeon_Ergodeon_*_structure.md - тут прописана структура файлов проекта

Редкие термины из науки и философии

Ergodeon - от "ergodic" (эргодический процесс) + окончание "-on". Нигде не используется как название проекта
Noumenon - философский термин Канта, "вещь в себе", скрытая сущность за интерфейсом
Syntaxis - греческое "syn-taxis" (упорядочивание), отражает суть планирования и структуризации
Составные неологизмы

Planwright - plan + wright (мастер), мастер планов. Не встречается как название
Codewright - аналогично, мастер кода
Logosync - logos (разум) + sync (синхронизация). Синхронизация замысла и кода
Cognivault - cognition + vault, хранилище когнитивных процессов (память + планирование)
Редкие слова

Palimpsest - пергамент с многократной перезаписью, метафора итеративной разработки с ревью
Tessera - элемент мозаики, каждый шаг пайплайна как фрагмент целого
Armillary - армиллярная сфера (астрономический инструмент), система орбит/циклов
Скандинавские / финские редкие

Runemark - руна + mark, знак-инструкция
Seidr - древнескандинавская практика предвидения и влияния на события
Verdandi - норна настоящего (одна из трех норн, определяющих судьбу), редко используется
